# A-VISS

Automated Vital Intelligence for Stress Surveillance (A-VISS) is a prototype dashboard developed as part of an undergraduate thesis in Information Systems at Hasanuddin University.

The system utilizes a Subject-Dependent CNN-LSTM model to classify nurse stress levels based on multimodal physiological sensor data, including:

- Heart Rate (HR)
- Electrodermal Activity (EDA)
- Skin Temperature (TEMP)
- Accelerometer Signals (X, Y, Z)

## Features

- Stress level classification
- Physiological signal visualization
- Stress score monitoring
- Interactive Streamlit dashboard
- Model performance overview

## Technology Stack

- Python
- TensorFlow / Keras
- Scikit-Learn
- Streamlit
- Plotly

## Best Model Performance

- Accuracy: 85%
- Weighted F1-Score: 86%
